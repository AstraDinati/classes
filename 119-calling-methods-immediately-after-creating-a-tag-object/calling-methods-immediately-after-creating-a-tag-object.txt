Вызов методов сразу после создания объекта Tag
Пусть с помощью нашего класса Tag мы создаем инпут:

<?php
	$tag = new Tag('input');
	
	echo $tag
		->setAttr('id', 'test')
		->setAttr('class', 'eee bbb')
		->open();
?>
Можно не создавать отдельную переменную для объекта, а создать его на месте и сразу начать применять к 
нему цепочку методов:

<?php
	echo (new Tag('input'))
		->setAttr('id', 'test')
		->setAttr('class', 'eee bbb')
		->open();
?>
Такое удобно в том случае, когда создаваемый тег уникальный и нужен нам только в одном месте. В этом 
случае мы не будем плодить лишних переменных.
