Дополнительные методы класса Tag
Наведем некоторый марафет нашему классу, добавив дополнительные публичные методы, которые пригодятся нам в 
дальнейшем.

№1⊗ppOpClTgAM

Реализуйте геттер getName, возвращающий название нашего тега (то есть значение свойства name).

№2⊗ppOpClTgAM

Реализуйте геттер getText, возвращающий текст нашего тега (то есть значение свойства text).

№3⊗ppOpClTgAM

Реализуйте геттер getAttrs, возвращающий массив всех атрибутов тега (то есть значение свойства attrs).

№4⊗ppOpClTgAM

Реализуйте геттер getAttr, параметром принимающий имя атрибута и возвращающий значение этого атрибута (а если 
такого атрибута нет - то null).
